This is a simple responsive webpage with a card layout and a styled button with following specifications.

Create a responsive webpage using HTML and CSS.
The page should include:
- A heading, a paragraph, and a button in the center of the page.
- A card layout with a shadow effect around the content.

Style the button to:
- Have a blue background, white text, and a subtle shadow.
- On hover, change the background color to green and scale it slightly larger.

Ensure the design is responsive by:
- Centering the content vertically and horizontally on larger screens.
- On smaller screens, ensuring the card takes up 90% of the screen width.
